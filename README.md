# DeepSeek Bulk Delete

A Chrome extension to bulk delete conversations on [chat.deepseek.com](https://chat.deepseek.com).

## Usage

1. Load the extension unpacked at `chrome://extensions/` (enable Developer mode)
2. Navigate to [chat.deepseek.com](https://chat.deepseek.com) and log in
3. Click the extension icon and use **Add Checkboxes** to add selection boxes to your conversations
4. Select conversations (hold Shift for range selection) and click **Bulk Delete**

## Manual Install (before store approval)

1. Download or clone this repo
2. Open `chrome://extensions/` and enable **Developer mode**
3. Click **Load unpacked** and select the folder containing these files
